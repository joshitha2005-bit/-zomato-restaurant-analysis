 # Zomato Restaurant Analysis 🍽️

Exploratory data analysis of 9500+ Zomato restaurants across India.

## Key Findings
- New Delhi has the most restaurants (5000+)
- Most restaurants are rated between 3.0 and 3.5
- 72% of restaurants don't offer online delivery
- North Indian and Chinese are the most popular cuisines
- Higher priced restaurants tend to have better ratings

## Tools Used
- Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

## Dataset
Zomato Restaurants Dataset from Kaggle
